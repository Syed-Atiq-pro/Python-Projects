### ЁЯзо Project 01 тАФ Calculator

![Python](https://img.shields.io/badge/Python-Basics-3776AB?style=flat-square&logo=python&logoColor=white)
![Concepts](https://img.shields.io/badge/Concepts-Functions%20%7C%20Conditions%20%7C%20Loops-orange?style=flat-square)

**What it does:**
A clean command-line calculator that performs addition, subtraction, multiplication, and division with proper error handling (like divide-by-zero protection).

**What I learned building this:**
- Writing and calling functions
- Using if/elif/else conditions
- Handling user input safely
- Basic error handling with try/except

```python
def calculator():
    print("=" * 30)
    print("   ЁЯзо Simple Python Calculator")
    print("=" * 30)
    
    num1 = float(input("Enter first number: "))
    op   = input("Enter operator (+, -, *, /): ")
    num2 = float(input("Enter second number: "))

    if op == '+':   result = num1 + num2
    elif op == '-': result = num1 - num2
    elif op == '*': result = num1 * num2
    elif op == '/':
        if num2 == 0:
            return print("тЭМ Error: Cannot divide by zero!")
        result = num1 / num2
    else:
        return print("тЭМ Invalid operator!")

    print(f"\nтЬЕ Result: {num1} {op} {num2} = {result}")

calculator()
```

### ЁЯО▓ Project 02 тАФ Number Guessing Game

![Python](https://img.shields.io/badge/Python-Basics-3776AB?style=flat-square&logo=python&logoColor=white)
![Concepts](https://img.shields.io/badge/Concepts-Random%20%7C%20Loops%20%7C%20Logic-orange?style=flat-square)

**What it does:**
The computer picks a secret number between 1тАУ100. You get hints (higher/lower) and limited attempts to guess it.

**What I learned building this:**
- Using the `random` module
- while loops with conditions
- Giving user feedback based on logic
- Counting attempts

```python
import random

def guessing_game():
    secret = random.randint(1, 100)
    attempts = 0
    max_tries = 7

    print("ЁЯО▓ I'm thinking of a number between 1 and 100!")
    print(f"   You have {max_tries} attempts. Good luck!\n")

    while attempts < max_tries:
        guess = int(input(f"Attempt {attempts+1}/{max_tries} тЖТ Your guess: "))
        attempts += 1

        if guess < secret:
            print("   ЁЯУИ Too LOW! Go higher.\n")
        elif guess > secret:
            print("   ЁЯУЙ Too HIGH! Go lower.\n")
        else:
            print(f"   ЁЯОЙ CORRECT! You got it in {attempts} attempt(s)!")
            return

    print(f"   ЁЯШФ Game Over! The number was {secret}.")

guessing_game()
```

### ЁЯУЛ Project 03 тАФ To-Do List App

![Python](https://img.shields.io/badge/Python-Intermediate-3776AB?style=flat-square&logo=python&logoColor=white)
![Concepts](https://img.shields.io/badge/Concepts-Lists%20%7C%20Functions%20%7C%20Menu%20Logic-orange?style=flat-square)

**What it does:**
A terminal-based to-do list where you can add tasks, view all tasks, mark them complete, and delete them.

**What I learned building this:**
- Managing lists dynamically
- Building a menu-driven program
- Using functions to keep code clean
- Real-world CRUD logic (Create, Read, Update, Delete)

```python
tasks = []

def show_tasks():
    if not tasks:
        print("ЁЯУн No tasks yet! Add one.")
    else:
        print("\nЁЯУЛ Your Tasks:")
        for i, task in enumerate(tasks, 1):
            status = "тЬЕ" if task["done"] else "тП│"
            print(f"   {i}. {status} {task['name']}")

def add_task():
    name = input("Enter task: ")
    tasks.append({"name": name, "done": False})
    print(f"   тЬЕ '{name}' added!")

def complete_task():
    show_tasks()
    idx = int(input("Enter task number to complete: ")) - 1
    tasks[idx]["done"] = True
    print("   ЁЯОЙ Task marked complete!")

def delete_task():
    show_tasks()
    idx = int(input("Enter task number to delete: ")) - 1
    removed = tasks.pop(idx)
    print(f"   ЁЯЧСя╕П '{removed['name']}' deleted!")

# Main menu loop
while True:
    print("\n" + "="*30)
    print("  ЁЯУЛ TO-DO LIST тАФ MAIN MENU")
    print("  1. View Tasks")
    print("  2. Add Task")
    print("  3. Complete Task")
    print("  4. Delete Task")
    print("  5. Exit")
    choice = input("\n  Choose (1-5): ")

    if choice == "1":   show_tasks()
    elif choice == "2": add_task()
    elif choice == "3": complete_task()
    elif choice == "4": delete_task()
    elif choice == "5": print("ЁЯСЛ Goodbye!"); break
    else: print("тЭМ Invalid choice.")
```

### ЁЯФР Project 04 тАФ Password Strength Checker

![Python](https://img.shields.io/badge/Python-Intermediate-3776AB?style=flat-square&logo=python&logoColor=white)
![Concepts](https://img.shields.io/badge/Concepts-String%20Methods%20%7C%20Logic%20%7C%20Regex-orange?style=flat-square)

**What it does:**
Checks how strong your password is тАФ Weak, Medium, or Strong тАФ based on length, uppercase, numbers, and special characters.

**What I learned building this:**
- String methods (`.isupper()`, `.isdigit()`, etc.)
- Writing scoring logic
- Using `any()` with conditions

```python
def check_password(password):
    score = 0
    feedback = []

    if len(password) >= 8:
        score += 1
    else:
        feedback.append("тЭМ At least 8 characters needed")

    if any(c.isupper() for c in password):
        score += 1
    else:
        feedback.append("тЭМ Add uppercase letters")

    if any(c.isdigit() for c in password):
        score += 1
    else:
        feedback.append("тЭМ Add numbers")

    if any(c in "!@#$%^&*()" for c in password):
        score += 1
    else:
        feedback.append("тЭМ Add special characters (!@#$...)")

    levels = {4: "ЁЯЯв STRONG", 3: "ЁЯЯб MEDIUM", 2: "ЁЯЯа WEAK", 1: "ЁЯФ┤ VERY WEAK"}
    strength = levels.get(score, "ЁЯФ┤ VERY WEAK")

    print(f"\nЁЯФР Password Strength: {strength}")
    if feedback:
        print("ЁЯТб Suggestions:")
        for tip in feedback:
            print(f"   {tip}")

password = input("Enter your password: ")
check_password(password)
```
